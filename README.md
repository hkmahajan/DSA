# Qspiders Java Programs

This repository contains various Java programs covering foundational concepts, data structures, and algorithms. It serves as a comprehensive practice and reference guide for Java programming.

## Directory Structure

- **Arrays**: Programs related to array manipulations, subarrays, prefix sums, and pair finding.
- **Binary_Search**: Implementations of Binary Search and related problems (e.g., finding occurrences, floor/ceil, square root).
- **Collection_Framework**: Examples using Java Collections like ArrayList.
- **ExecptionHandling**: Examples of checked and unchecked custom exceptions and general exception handling.
- **LinkedList**: Implementations of Singly and Doubly Linked Lists.
- **Linear_Search**: Linear search implementation, including recursive approaches.
- **Number_Programs**: Mathematical and number-based logic programs (e.g., Armstrong, Fibonacci, Prime, Palindrome).
- **OOPs**: Object-Oriented Programming concepts including Abstraction, Encapsulation, Inheritance, and Polymorphism.
- **Pattern_Programs**: Various printing patterns like triangles, quadrilaterals, asterisks, and number patterns.
- **Reccrusion**: Basic programs illustrating recursion.
- **Sorting_Algorithms**: Implementations of basic sorting algorithms like Bubble Sort and Selection Sort.
- **Stack**: Stack implementation using arrays.
- **String**: String manipulation programs such as palindromes and word reversal.

## How to Run

1. Make sure you have the Java Development Kit (JDK) installed on your system.
2. Compile any program using: javac FileName.java
3. Run the compiled program using: java FileName
